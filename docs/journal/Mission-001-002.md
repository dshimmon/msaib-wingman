## Key Lessons

- Git tracks the history of a project rather than simply storing files.
- GitHub is a remote copy of the repository, not Git itself.
- Every project should have its own isolated Python environment.
- Generated files should not be committed to source control.
- The editor is where software is written.
- The terminal is where software is executed.
- Every meaningful milestone deserves its own commit.
